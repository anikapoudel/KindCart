# KindCart

**A cross-platform web and mobile application for thrift shopping and donations.**

KindCart enables users to buy and sell second-hand items while facilitating donations of useful goods to those in need, promoting sustainable consumption and social responsibility.

---

## Technology Stack

| Technology              | Purpose                                 |
| ----------------------- | --------------------------------------- |
| Flutter                 | Cross-platform frontend (Web + Android) |
| Firebase Authentication | User login and role management          |
| Cloud Firestore         | Database                                |
| Firebase Storage        | Image uploads                           |
| Dialogflow              | AI chatbot                              |
| Provider                | State management                        |

---

## Key Features

* **4 User Roles** – Buyer, Seller, Donor, Admin
* **Thrift Shopping** – Browse, search, filter, cart, wishlist
* **Donations** – Upload items, admin approval, impact dashboard
* **Chat with QR Payment** – Real-time messaging with payment proof exchange
* **AI Chatbot** – Dialogflow-powered assistant
* **Admin Panel** – User management, product moderation, donation approval

---

## Getting Started

### Prerequisites

* Flutter SDK 3.16+
* Firebase account

### Installation

```bash
git clone https://github.com/anikapoudel/KindCart.git
cd kindcart
flutter pub get
flutter run
```
